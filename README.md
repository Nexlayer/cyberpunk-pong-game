# Cyberpunk Pong

A futuristic, neon-styled Pong game with cyberpunk aesthetics and visual effects.

## Features

- **Cyberpunk Visual Design**: Neon colors, glowing effects, and futuristic UI
- **Particle Effects**: Dynamic particle explosions on ball collisions
- **Trail Effects**: Ball leaves a glowing trail as it moves
- **Animated Background**: Moving grid lines and scan effects
- **AI Opponent**: Intelligent AI that adapts difficulty based on score
- **Responsive Design**: Works on desktop and mobile devices
- **Smooth Animations**: 60fps gameplay with fluid motion

## Controls

- **W/S Keys**: Move your paddle up/down
- **Spacebar**: Start/pause the game
- **Mouse**: Click buttons for game control

## Game Mechanics

- Classic Pong gameplay with cyberpunk styling
- Ball bounces off paddles and walls
- Score tracking for both player and AI
- AI difficulty increases as the game progresses
- Visual feedback with particles and glow effects

## Technologies Used

- HTML5 Canvas for rendering
- CSS3 for styling and animations
- Vanilla JavaScript for game logic
- Google Fonts (Orbitron) for typography

## How to Play

1. Open `index.html` in a modern web browser
2. Click "START GAME" or press Spacebar
3. Use W and S keys to control your paddle
4. Try to beat the AI opponent!
5. Press Spacebar to pause/resume
6. Click "RESET" to start a new game

## Visual Effects

- **Neon Glow**: All game elements have glowing effects
- **Particle System**: Collisions create colorful particle explosions
- **Trail System**: Ball leaves a fading trail
- **Background Animation**: Moving grid and scan lines
- **Gradient Effects**: Paddles have gradient fills
- **Dynamic Lighting**: Real-time shadow and glow effects

## Browser Compatibility

- Chrome (recommended)
- Firefox
- Safari
- Edge

Requires a modern browser with HTML5 Canvas support.

## Development

The game is built with vanilla JavaScript and doesn't require any build tools or dependencies. Simply open the HTML file in a browser to play.
